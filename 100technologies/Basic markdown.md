## **Basic Structure**

- Markdown is plain text that turns into formatted notes.
    
- **Blank lines** separate paragraphs. [Obsidian Help](https://help.obsidian.md/syntax)
    

---

## **Headings**

Use `#` for section titles:

`# Heading 1 ## Heading 2 ### Heading 3`

(Up to 6 levels) [Obsidian Help](https://help.obsidian.md/syntax)

---

## **Text Formatting**

| Style                                                           | Syntax                   |
| --------------------------------------------------------------- | ------------------------ |
| **Bold**                                                        | `**text**` or `__text__` |
| _Italic_                                                        | `*text*` or `_text_`     |
| ~~Strikethrough~~                                               | `~~text~~`               |
| ==Highlight==                                                   | `==text==`               |
| (Works inline) [Obsidian Help](https://help.obsidian.md/syntax) |                          |

## **Quotes & Callouts**

- A quote block:
    
    `> This is a quote`
    
- **Callout:** add a type inside:
    
    `> [!info] This is an info callout ``` :contentReference[oaicite:4]{index=4}`
    

---

## **Lists**

- **Unordered:** `-`, `*`, or `+`
    
- **Ordered:** `1.`, `2)` etc.
    
- **Tasks:** `- [ ]` or `- [x]`
    

`- [ ] Task 1. Item`

Supports **nested lists** via indenting. [Obsidian Help](https://help.obsidian.md/syntax)
Examples:

- [x] Tarea 1
- [ ] Tarea 2

---

## **Code & Syntax Highlighting**

- **Inline code:** `` `code` ``
    
- **Code block:**
    
    ` ```lang code here `
    

You can use backticks or tildes for fences. [Obsidian Help](https://help.obsidian.md/syntax)
