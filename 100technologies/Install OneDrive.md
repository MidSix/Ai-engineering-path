## Install and Set Up OneDrive

1. Download OneDrive from:
   https://www.microsoft.com/onedrive (If link doesn't work just google OneDrive Download)

2. Install and sign in with your Microsoft account.

3. During setup:
   - Choose the default OneDrive folder location  
     (usually `C:\Users\<YourName>\OneDrive`)
   - Enable syncing

After setup, you should have a **OneDrive folder** on your computer. This folder is the one that's gonna be used to sync your files. All files inside that OneDrive folder are upload in the cloud.
